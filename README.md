## Publishing Notes

- Static hosting is enough: Netlify, Vercel, GitHub Pages, Cloudflare Pages, or any cPanel hosting.
- Replace the placeholder production domain `https://aevum-house.com` in `sitemap.xml` and all Open Graph tags before going live with the final domain.
- Replace the sample email and WhatsApp number with the final house contact details.
- Update collection imagery as real stock changes.
